How This Page Works

Where to press

The roster table above is the working area. Coaches should use the last column and press Write Report for each player.

What to add

Each student gets a written recap and an optional proof video. Skill scoring only happens once every six months inside the progress report.

When it appears

Once submitted, the report appears in this session record, the member profile, and any later progress report.
