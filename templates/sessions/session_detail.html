How This Page Works

Where to press

The roster table above is the working area. Coaches should use the last column and press Write Report for each player.

What to add

Each student can get a written recap plus an optional proof video. That is the exact post-session follow-up saved into the student history.

When it appears

Once submitted, the report appears in this session record, the member profile, and any later progress report that references recent session reports.
